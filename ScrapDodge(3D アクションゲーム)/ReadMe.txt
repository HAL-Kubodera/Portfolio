SCRAP DODGE ReadMe

[概要]
ジャンル : 3Dアクション
制作人数 : 一人
制作期間：４か月
使用スキル : Directx11、C++言語、VisualStudio、GitHub、Maya

[操作方法]
-ゲーム中-
移動 : WASD
ジャンプ : SPACE
スロウモード : 右クリック長押し
コウゲキ : 相手が疲れた状態(敵からパーティクルが出ているとき)に左クリック
ポーズモード : P
-その他-
ボタンの選択 : 矢印キー
ボタンの決定 : ENTER
チュートリアルなどの早送り : ENTER

[ゲーム概要]
「避ける」に特化したアクションゲームです。
相手の攻撃を避け続けて最後に大きな一撃を叩きこみましょう。
相手の体力が減るごとに攻撃は強くなります。

[技術的アピールポイント]
今回の制作ではコンポーネント指向やステートマシンなどのコード設計に挑戦しました。
またたくさんの攻撃を描画しても重くならないようにDrawCallを減らしたり、なるべく処理を使いまわしたりすることで軽量化を図りました。
